Own the game. We ship engines.

Holowan Underlabs is unofficial fan work. Engines and tools are open; retail game data is not ours to host or redistribute.

FAQ

  • Do I need KotOR / TSL? Yes. Holowan and KotOR.js replace the program, not the content. You must legally own the PC game and supply its files yourself.
  • Do you distribute game data? No. We do not host art, archives, audio, or other copyrighted game assets. We do not condone piracy.
  • Affiliated with Lucasfilm / BioWare / Obsidian / Aspyr? No.
  • Which browser? Desktop Chrome or Edge for the directory picker used by KotOR.js.
  • Tor? No onion is published. Treat any claimed .onion as unverified.
  • Need to buy the game? See Need KotOR? on the home page.

Project Borealis

Project Borealis is a private research reference involving proprietary binary / platform materials related to KotOR II. It is not open source, not a clean-room reimplementation, and not distributed here.

Do not request Borealis sources or dumps. Public work (KotOR.js, reone, tools) must be independent and must not redistribute copyrighted game code or assets.
